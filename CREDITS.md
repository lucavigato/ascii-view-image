# Credits

This project is an enhanced fork of the open-source ASCII art generator by **Gouws Xander**.
Original Repository: [https://github.com/gouwsxander/ascii-view](https://github.com/gouwsxander/ascii-view)

## Contributors
- **Gouws Xander**: Original author, core logic for Sobel edge detection and ASCII conversion.
- **Luca**: Architecture refactoring, PNG export implementation, scaling logic, "Pixel Perfect" mode, and build system overhaul.

## Open Source Libraries Used
- **stb_image** (Sean Barrett): Public domain image loading library.
- **Cairo**: 2D graphics library.
- **Pango**: Text layout and rendering library.
